# cra-template-vite

`cra-template-vite` 用于快速生成开发项目模板

# 安装和使用

1. 查看[私有 npm 仓库使用](https://newrank.yuque.com/nrcd/hpdymk/rwmdssvx5dpzmxv0?singleDoc#)

2. 命令行安装

```shell
npx cra-template-vite my-app
```

**安装一个确定的模板**

```shell
npx cra-template-vite my-app --template mobile
```

**现有模板参数**

|  参数  |          描述          |
| :----: | :--------------------: |
| mobile | 生成移动端开发项目模板 |
